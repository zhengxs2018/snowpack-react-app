## 开发步骤

你需要安装 Node.js 的版本为 12+.

克隆此仓库后运行:

```bash
# 安装依赖
$ yarn install

# 本地开发
$ yarn dev

# 构建代码
$ yarn build
```

在 package.json 文件的 scripts 部分还有一些其他脚本可用.

## 运行单元测试

```bash
# 单元测试
$ npm test
```
